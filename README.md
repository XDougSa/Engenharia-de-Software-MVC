# Conecte-se: desenvolvimento web para inclusão digital

## **Coordenador**: Professor Doutor Edeilson Milhomem da Silva  

A internet tem se tornado uma ferramenta essencial para o desenvolvimento pessoal e profissional. A internet viabiliza a criação e o gerenciamento de conteúdo online, habilidades que se tornam cada vez mais requisitadas no mercado de trabalho. Nesse contexto, o conhecimento em HTML (HyperText Markup Language), CSS (Cascading Style Sheets) e PHP (Hypertext Preprocessor) se destaca como fundamental para aqueles que desejam construir e personalizar páginas da web, seja em nível básico ou avançado.

O domínio de HTML, CSS e PHP habilita oportunidades profissionais na área de desenvolvimento web, além de oferecer capacitação para participantes desenvolverem suas próprias plataformas online, impusionando assim naturalmente iniciativas empreendedoras, projetos pessoas e a presença digital. Aacessibilidade desses conhecimentos permite que pessoas de diversas áreas profissionais ampliem suas competências digitais, o que é especialmente relevante em um cenário de constantes mudanças e adaptação às novas tecnologias.

Este projeto de Treinamento em HTML, CSS e PHP se justifica pela necessidade de impulsionar a formação profissional e capacitada em desenvolvimento web e também pelo potencial empreendimento digital que as habilidades desenvolvida podem proporcionar.  Ao oferecer um curso estruturado, com conteúdos que abrangem desde os conceitos básicos até técnicas mais avançadas, preten-se preparar os participantes para que possam atuar de forma autônoma na construção de websites, atendendo tanto às necessidades pessoais quanto às demandas do mercado.

Outro fator relevante que justifica o projeto é a redução das barreiras de entrada para aqueles que desejam ingressar na área de tecnologia, uma vez que propiciará um ambiente de aprendizagem inclusivo e acessível. Assim, o projeto visa contribuir para a diminuição das desigualdades digitais e promover o desenvolvimento de competências que são cada vez mais valorizadas no mundo contemporâneo. Destaca-se que este é um projeto vinculado ao Programa EPIC (Engajar, Praticar, Integrar e Colaborar) para a creditação de Extensão nas disciplinas de Práticas Extensionistas I, II e III do curso de Ciência da Computação, Câmpus-Palmas-UFT.

---

O presente projeto contempla um treinamento online sobre HTML, CSS, JavaScript e PHP com o propósito de habilitar os participantes a desenvolverem aplicações web com PHP, por meio da aplicação da arquitetura MVC (Model-View-Controller).

## Aulas

### [Aula 1 - Introdução Geral: Panorama e Objetivos do Curso](https://www.youtube.com/watch?v=vd1XtgypvpE)
#### Apresentadora: Fernanda
Nesta aula, será apresentada uma visão geral do minicurso de engenharia desoftware com foco na arquitetura MVC. Vamos discutir os principais objetivos do
curso, que incluem a compreensão dos fundamentos da arquitetura MVC e a aplicação desses conceitos no desenvolvimento de projetos reais. Serão abordados também os tópicos que serão explorados em cada aula, o cronograma, e os recursos disponíveis. A introdução servirá para alinhar as expectativas dos alunos e preparar o terreno para a aprendizagem dos conceitos e práticas que serão desenvolvidos ao longo do curso.

### [Aula 2 - Configurações Essenciais: Preparando o Ambiente de Trabalho](https://www.youtube.com/watch?v=N22vkM_hrew)
#### Apresentador: Douglas
Nesta aula, o aluno aprenderá como preparar o ambiente de trabalho no que se concerne a instalação da IDE, que, no caso, será o Visual Studio CODE; a instalação da linguagem PHP e o gerenciador de banco de dados “PostgreSQL”.

### [Aula 3 - Explorando o HTML: Fundamentos e Estrutura](https://www.youtube.com/watch?v=TPfYDU3ao4Y)
#### Apresentadora: Fernanda
Nesta aula, exploraremos os fundamentos e a estrutura do HTML, a linguagem de marcação fundamental para a criação de páginas web. Discutiremos os principais
elementos e tags HTML, suas funções e como utilizá-los para construir a estrutura básica de uma página. Serão abordados conceitos como elementos, atributos e a
hierarquia de documentos. Esta aula fornecerá a base necessária para a construção de páginas web e preparará os alunos para a aplicação de estilos e scripts nas aulas seguintes.

### [Aula 4 - Introdução ao CSS: O Que é e Como Utilizá-lo](https://www.youtube.com/watch?v=FosqU8Ymf-E)
#### Apresentadora: Larissa
A Aula 4 do minicurso oferecerá uma Introdução ao CSS, explicando o que é e como utilizá-lo no desenvolvimento web. Serão abordados os conceitos fundamentais de
CSS, incluindo seletores, propriedades e a aplicação de estilos em elementos HTML. A aula demonstrará como criar layouts básicos e personalizar a aparência de
páginas web. Ao final, será possível entender como o CSS interage com o HTML para construir interfaces visuais.

### [Aula 5 - Introdução ao JavaScript: Tornando suas Páginas Dinâmicas](https://www.youtube.com/watch?v=bnWFMD2KQ8E)
#### Apresentador: Iago
Nessa aula o aluno será introduzido a linguagem de programação Javascript e de seus usos. Ele será contextualizado nos usos que a linguagem possui no mundo
Web, além de ser convidado a aprender o básico da linguagem e a fazer operações básicas com a linguagem usando de um interpretador online.

### [Aula 6 - Explorando PHP: Fundamentos e Primeiros Passos](https://www.youtube.com/watch?v=DfJzIO_OYtc)
#### Apresentadora: Larissa
A Aula 6 do minicurso focará nos Fundamentos Básicos do PHP, abordando conceitos essenciais como variáveis, operadores e estruturas de controle. Serão
apresentados exemplos simples para demonstrar a lógica básica por trás do código PHP. A aula é voltada para iniciantes, proporcionando os primeiros passos no
desenvolvimento web com PHP. Ao final, será possível criar scripts básicos e compreender a estrutura da linguagem.

### [Aula 7 - Configuração do Git: Iniciando o Controle de Versão](https://www.youtube.com/watch?v=WgEAWOw26zE)
#### Apresentador: João
Nesta aula mostraremos como configurar o git localmente, com o usuário podendo se conectar a ele e entendendo um pouco sobre como ele funciona e para que é
utilizado na prática.

### [Aula 8 - Iniciando seu Projeto: Estruturação e Preparação](https://www.youtube.com/watch?v=owkaEXaBCls)
#### Apresentador: Iago
Durante essa aula será feita e explicada a organização do projeto que irá ser feito durante as próximas aulas. Serão explorados conceitos como organização de
projetos em PHP, configurações iniciais de repositório, uso do arquivo .gitignore, uso de tokens de JWT com implementação de função para leitura, assim como variáveis de ambiente e seu uso como forma de garantir maior segurança ao projeto.

### [Aula 9 - Iniciando com Banco de Dados: Como Configurar e Conectar](https://www.youtube.com/watch?v=O_OWWgVHtCs)
#### Apresentador: Douglas
Nesta aula, será abordada a integração do banco de dados PostgreSQL com uma aplicação em padrão MVC. O aluno aprenderá a configurar a conexão com o banco
pelo componente “Model”, definindo parâmetros como host, porta, nome do banco, usuário e senha; além das informações que devem ser fornecidas no arquivo .env.

### [Aula 10 - Autenticação com JWT: Como Criar um Sistema de Login Seguro](https://www.youtube.com/watch?v=Ri422xLsx_4)
#### Apresentadora: Larissa
A Aula 10 do minicurso abordará a Autenticação com JWT (JSON Web Token), ensinando como criar um sistema de login seguro utilizando o padrão View-Model-Controller. O processo incluirá a solicitação de login na View, a validação dos dados no Model, e a geração e verificação dos tokens JWT no Controller.

### [Aula 11 - Criando a Tela de Cadastro: Estrutura e Implementação](https://www.youtube.com/watch?v=WZdxNB0SOVY)
#### Apresentadora: Fernanda
Nesta aula, vamos focar na criação da tela de cadastro, abordando tanto a estrutura quanto a implementação. Serão discutidos os elementos necessários para projetar
uma interface de cadastro, incluindo campos de entrada e layout. Exploraremos como integrar a tela de cadastro com a lógica de backend para armazenar e gerenciar as informações dos usuários. Esta aula permitirá aos alunos aplicar os conceitos aprendidos anteriormente para desenvolver uma funcionalidade essencial
em aplicações web, garantindo que a tela de cadastro seja funcional.

### [Aula 12 - Implementação de Dashboard: Consolidando Dados em uma Visualização Única](https://www.youtube.com/watch?v=geVW0jFhv9I)
#### Apresentador: João
Nesta aula teremos a visualização de todos os usuários cadastrados na tela de quem conseguiu acessar o sistema. Teremos a oportunidade de trabalhar com componentização, java script e com as operações de delete e put.
